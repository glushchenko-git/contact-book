Книга контактов - React приложение

Описание приложения

Приложение "Книга контактов" представляет собой удобный инструмент для управления списком контактов. Пользователи могут добавлять новые контакты с именем и номером телефона, просматривать существующие контакты и удалять ненужные записи.

Основные возможности

Добавление новых контактов (имя и телефон)
Просмотр списка всех контактов
Удаление контактов из списка
Валидация вводимых данных
Локальное хранение данных в браузере
Особенности реализации

Использование функциональных компонентов React с хуками (useState, useEffect)
Локальное состояние для управления списком контактов и формой ввода
Валидация формы перед добавлением контакта
Сохранение данных в localStorage для сохранения состояния между сессиями
Адаптивный интерфейс с использованием CSS Grid/Flexbox
Технологический стек

React 18+
CSS3 (модули или styled-components)
LocalStorage API
Структура компонентов

App - корневой компонент
ContactForm - форма добавления нового контакта
ContactList - список контактов
ContactItem - отдельный элемент контакта
Установка и запуск

bash
# Клонирование репозитория
git clone <repository-url>

# Установка зависимостей
npm install

# Запуск приложения
npm start
